About

Hello world! My name is Brenton Gibbons and breakdesign.com.au is my showcase website. I am a tinkerer, designer and aspiring engineer that is always trying to learn new things. I am a qualified designer, with a passion for quality design through clean code. Designs with smart CSS, beautiful SVGs and subtle JS are my favourite.

Currently, I am studying Engineering and Medical Science. I believe that learning is a lifelong habit. The list of topics I want to learn is always growing. I have developed a strong love for IoT recently, and am always tinkering with a new project. I also love being in the kitchen, trying new recipes with varying levels of success.

breakdesign.com.au is my site to showcase what personal project I am currently working on and future plans. At the moment, it's the next iteration of this site.

Projects

My current project is learning new skills to develop a increasingly more awesome version of breakdesign.com.au .

Over the last few months, I have been using an AGILE methodology to develop this site, while learning new skills at the same time. I have used AGILE strategies in the past to help with project management, and have found it a amazing tool for development. Whereas previously I would have taken months to build a new site, I had the first iteration of this site out within a couple of weeks. The idea of iteration has become my new best friend

In future iterations, I will be developing more new skills to put finer details into the site. I have developed a roadmap of new features to build into the site, inclduing:

  • GIT and Grunt.js automation
  • SVG and CSS Animation for Hero Images
  • SVG minify automation for optimised SCG code
  • Javascript controls to change CSS elements
  • ARIA code for screen reader support

The next sprint after this release will be mainly focused on completing the SVG components. It has been great to learn more about SVGs while developing this site. When I designed the previous version of the site, SVG was in its infancy and basically no support from any browsers. It has now become my favourite tool!
