PROJECTS

My current project is learning new skills to develop a increasingly awesome version of breakdesign.com.au.

Over the last few months, I have been using an AGILE methodology to develop this site, while learning new skills at the same time. Previously, I have used AGILE to help with project management, and have found it a amazing strategy for development. Whereas previously I would have taken months to build a new site, I had the first iteration of this site out within a couple of weeks. The idea of iteration has become my new best friend.

In future releases, I will be developing more new skills to put finer details into the site. My roadmap includes:

  • GIT and Grunticon automation
  • SVG and CSS Animation for Hero Images
  • SVG minify automation for optimised SVG code
  • Javascript controls to change CSS elements
  • ARIA code for screen reader support

The next release will be focused on completing the SVG components. It has been great to learn more about SVGs while developing this site.
